# SignIn Page - React Frontend

A user registration interface built with React, connected to a custom backend API.
[View Server Projetc] (https://github.com/emilyzugel/usersServer/)

## Features

- User registration with name, email, and age
- Real-time user list display
- Delete users functionality
- Responsive design

## Tech Stack

**Frontend:** React, Axios, Font Awesome, CSS3  
**Backend:** Node.js, Express, MongoDB, Mongoose

## API Integration

Connects to custom REST API with full <strong>CRUD operations</strong>.
<br>

> <italic>
  Ensure backend server is running on http://localhost:3000
</italic>
